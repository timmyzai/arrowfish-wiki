Arrowfish VPN 业务说明文档

整理日期：2026年4月 适用对象：BA · PM · 非技术团队成员 用途：产品上手参考 · PRD 素材

一、产品是什么

Arrowfish VPN 是一款面向中文用户的商业 VPN 服务，帮助用户安全访问 Google、YouTube、社交媒体等被中国防火长城屏蔽的境外网站。

组成部分说明
Arrowfish（现役）Android 应用（主力）+ Windows / macOS 客户端 + iOS 端 HiYou SSH（已冻结，开发者账号被封，无法更新）
FFF（下一代）正在开发的全新跨平台应用，覆盖 Android、iOS、PC，计划2026年6月上线
管理后台（BO）管理用户、订单、节点、内容的内部管理平台
商业模式：免费体验（有时间限制）→ 付费订阅（月 / 季 / 年）→ 高端专属服务

二、产品战略方向

产品当前策略
Arrowfish维持稳定，以 Bug 修复为主，不再投入新功能开发
FFF全力重构与测试，2026年6月正式发布
Arrowfish 与 FFF 架构不同（原生 Android vs Flutter 跨平台），在 Arrowfish 上做的新功能无法复用到 FFF，继续投入会造成资源浪费。

里程碑

时间目标
2026年4月下旬Arrowfish 上架 Google Play
2026年6月FFF 正式发布
2026年6月后Arrowfish 用户逐步引导迁移至 FFF
2026年7月起智能路由 + 加密货币支付网关开发

Arrowfish 与 FFF 用户数据独立，用户需主动切换，两款应用过渡期并存。FFF 将以欧盟注册公司独立运营。

六、VPN 连接功能

打开 App → 选择线路（或一键连接）→ 建立加密隧道 → 访问境外网站
线路类型说明可用用户
免费线路基础速度，有时间限制所有用户
标准线路稳定连接标准会员+
精品线路高质量，支持流媒体精品会员+
独享专线整条专线一人独用独享专线用户
专属IP固定出口IP地址专属IP用户

全球 156 个 VPN 节点，覆盖多国。连接建立超过 4 秒视为失败；手机锁屏超过 5 分钟无流量，服务端自动断连；修改账号权限后节点最多需要 60 秒同步。

⚠️防火长城封锁风险：中国大陆 GFW 会封锁 VPN 节点 IP、协议流量特征，乃至登录域名。封锁可无预警触发，且省级策略不同。这是节点不稳定投诉的根本原因，与节点服务商质量无关。

系统追踪用户注册来源渠道（Google广告、百度、视频推广等），可按渠道统计注册量与转化率。活动支持限时折扣、注册送券、购买赠时等形式，由管理员配置有效期和发放条件。

十六、合规与隐私

不记录可以保存
访问的网站、流量内容、连接了哪个节点、DNS 查询账号信息、订阅状态、支付记录、用户主动提交的客服内容
