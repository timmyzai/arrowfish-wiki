QA 测试手册 — 测试环境配置

更新日期：2026年4月13日 适合：QA 测试人员
ℹ️ 本文档仅适用于测试（Staging）环境。正式生产环境配置不同，上线前将另行发布正式环境参考文档。

一、测试环境基本信息

项目值
测试服务器52.230.7.201
后端 API 域名（客户端用）dev-client.o-faster.cc
管理后台域名dev-admin.o-faster.cc
数据库MySQL 8.0，与生产完全独立
测试节点端口1443（生产用 443）
ℹ️测试环境与生产环境数据库完全独立。测试账号不会出现在生产环境，反之亦然。

二、登录与账号安全

配置项当前值测试影响
密码错误上限5 次连续输错5次后账号锁定，不可无限重试
账号锁定时长30 分钟锁定后必须等30分钟才能再试
短信验证码长度6 位测试注册 / 找回密码时填6位
⚠️测试登录错误流程时，触发锁定后该账号30分钟内完全不可用。建议准备多个测试账号，避免影响测试进度。如需提前解锁，联系后端开发手动清除。

登录码（免密流程）说明

用户通过手机号注册后，系统生成 16位专属登录码（格式：4组×4字符）。这是用户下次登录的凭证，明文存储和展示。短信验证码（OTP）是一次性的，加密处理，两者完全不同。

三、免费用户时长限制

配置项测试环境生产环境测试影响
每日基础免费时长10 分钟30 分钟免费账号当天累计连线超过10分钟后自动断线
看广告奖励时长10 分钟/次30 分钟/次每看一次广告，免费时长+10分钟
每日最多看广告次数3 次2 次最多额外获得30分钟
测试环境每日上限40 分钟90 分钟—
ℹ️测试时长缩短（10分钟基础），是为了让 QA 更快触发"时长耗尽"边界场景。时长耗尽后，需等次日凌晨重置，或联系后端把该测试账号已用时长清零。

四、VPN 连接行为

配置项当前值测试影响
ConnIdle（服务端空闲断连）300秒（5分钟）手机锁屏后，超过5分钟无任何流量才触发服务端断连
握手超时4秒连接建立超过4秒视为失败
节点同步间隔60秒修改账号权限后，节点最多需要60秒才能感知变化
⚠️已知 Bug（待修复）：手机锁屏超过5分钟后，App 显示"已连线"，但境外网站打不开，延迟显示 65535ms。这是客户端 Doze 模式问题，与服务端无关。遇到请记录，不算测试环境异常。

八、支付功能

⚠️测试支付时，请使用最低金额（如¥1），避免产生不必要的真实扣款。
支付方式注意事项
微信支付 / 支付宝可测试，使用真实账号，选最低金额
USDT链上转账有真实手续费，请谨慎测试
Google Play 内购以实际测试包配置为准
Apple IAP使用沙盒测试账号

支付到账时间

  • 微信 / 支付宝：通常30秒内生效
  • USDT TRC20：最长等待 15 分钟
  • USDT ERC20：最长等待 30 分钟（链上确认时间）
⚠️已知 Bug（待修复）：支付成功、后端回调返回 200 后，App 界面不会自动刷新——用户仍看到未付款状态。需重新登录才能看到会员已到账。测试支付流程时请注意：支付成功≠App 显示成功，务必重新登录验证实际权限是否生效。

九、节点与线路同步机制

ℹ️关键：在后台修改用户会员等级或套餐权限后，不要立刻测试——需等待约 60 秒让节点同步，再验证权限是否生效。
操作生效延迟
修改用户会员等级约 60 秒
新增 / 修改线路约 60 秒
暂停 / 恢复节点约 60 秒
修改套餐价格即时（刷新页面）
